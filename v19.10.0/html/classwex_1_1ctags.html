static bool wex::ctags::find ( const std::string &  tag)
static

Find the tags matching ‘tag’, and fills the matches container.

Returns true if a matching tag is found, and calls frame open_file if name matches and there is no next match in another file. If the name is empty, next is invoked. Otherwise shows a dialog to select a file from the matches. Returns false if dialog was cancelled.

static void wex::ctags::open ( const std::string &  filename = DEFAULT_TAGFILE)

Opens ctags file.

The ctags file is obtained from the STCData associated with ex STC. Default uses standard ctags file, but you can choose your own name. This file is searched for in the current dir, and if not found in the config dir. You can also specify an absolute filename.
