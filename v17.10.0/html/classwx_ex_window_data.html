◆ Button()

wxExWindowData& wxExWindowData::Button ( long  button)

Sets buttons flags.

This is a bit list of the following flags:

  • wxOK
  • wxYES
  • wxAPPLY
  • wxSAVE
  • wxCLOSE
  • wxNO
  • wxCANCEL
  • wxHELP
  • wxNO_DEFAULT The default buttons are wxOK and wxCANCEL
