Detailed Description

Adds several File* methods to wxFile.

All the File* methods update the wxExStat member. Also takes care of synchronization, all you have to do is call CheckSync once in a while.

Constructor & Destructor Documentation

◆ wxExFile()

wxExFile::wxExFile ( bool  open_file = true)
inline

Default constructor.

The open_file parameter specifies the behaviour of FileLoad and FileSave, if true, the file is opened before calling DoFileLoad and DoFileSave, if false the file is not opened. That might be useful if you do not use the wxFile for loading and saving (as with XML documents), but still want to use the virtual file interface.
