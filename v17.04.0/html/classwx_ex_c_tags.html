bool wxExCTags::Find ( const std::string &  name) const

Find the tags matching `name'.

Returns true if a matching tag is found, and calls frame OpenFile if name matches and there is no next match in another file. Otherwise shows a dialog to select a file from the matches. Returns false if dialog was cancelled.
