Detailed Description

Offers an address range for vi (ex).

  • The range is derived from a number of lines,
  • or by a range string (including visual range for already selected text on the STC component). All methods return false if the range is not ok.

bool wxExAddressRange::Escape ( const std::string &  command)

Filters range with command.

The address range is used as input for the command, and the output of the command replaces the address range. For example: wxExAddressRange(96, 99).Escape("sort") or (ex command::96,99!sort) will pass lines 96 through 99 through the sort filter and replace those lines with the output of sort. Of course, you could also do: wxExAddressRange(96,99).Sort(). If you did not specify an address range, the command is run as an asynchronous process.
