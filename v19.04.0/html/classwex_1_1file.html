Detailed Description

Offers several methods to read / write files.

Also takes care of synchronization, all you have to do is call check_sync once in a while.

Constructor & Destructor Documentation

◆ file()

wex::file::file ( bool  open_file = true)

Default constructor.

The open_file parameter specifies the behaviour of file_load and file_save, if true, the file is opened before calling do_file_load and do_file_save, if false the file is not opened.
