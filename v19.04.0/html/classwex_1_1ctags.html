◆ ctags()

wex::ctags::ctags ( ex *  ex)

Constructor, opens ctags file.

Uses ex component for presenting ctags results. The ctags file is obtained from the STCData associated with ex STC. Default uses standard ctags file, but you can choose your own name. This file is searched for in the current dir, and if not found in the config dir. You can also specify an absolute filename.

bool wex::ctags::find ( const std::string &  tag)

Find the tags matching ‘tag’, and fills the matches container.

Returns true if a matching tag is found, and calls frame open_file if name matches and there is no next match in another file. If the name is empty, next is invoked. Otherwise shows a dialog to select a file from the matches. Returns false if dialog was cancelled.
