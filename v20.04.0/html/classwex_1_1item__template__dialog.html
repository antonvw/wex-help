Detailed Description

template<class T>
class wex::item_template_dialog< T >

Offers a dialog template to set several items.

If you only specify a wxCANCEL button, the dialog is readonly. When pressing the:

  • wxAPPLY button
  • wxOK, wxCANCEL button for a modeless dialog
  • a item::BUTTON
  • a item::COMMANDLINKBUTTON
  • a item::TOGGLEBUTTON the method frame::on_command_item_dialog is invoked.
